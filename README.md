# MSIS 522 — HW1: COVID-19 Mortality Risk Prediction

## Project Overview

End-to-end data science workflow for predicting COVID-19 patient mortality using anonymized medical records from approximately 1 million patients in Mexico. The project covers descriptive analytics, predictive modeling (five model families), SHAP-based explainability, and a deployed Streamlit dashboard.

**Target variable:** `DEATH` (binary — 0 = Survived, 1 = Died)

## Dataset

- **Source:** General Directorate of Epidemiology, Mexican Ministry of Health (via instructor-provided Google Drive link)
- **Raw size:** ~1,048,576 patients × 17 features
- **Working sample:** 10,000 patients (5,000 per class, balanced under-sampling)
- **Features:** Age, Sex, Hospitalized, COVID Positive, Pneumonia, Pregnancy, Diabetes, COPD, Asthma, Immunosuppression, Hypertension, Cardiovascular, Obesity, Renal Chronic, Tobacco, Other Disease

## Setup Instructions

### 1. Clone the repository

```bash
git clone <your-repo-url>
cd <repo-name>
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Run the notebook

Open and run all cells in `notebook/MSIS_522_HW1_COVID19_Complete.ipynb`. This will:
- Download the dataset to `data/covid.csv`
- Run all descriptive analytics and save plots to `outputs/`
- Train, tune, and evaluate all five models
- Save trained models and artifacts to `models/`
- Save the metrics comparison table to `outputs/model_metrics.csv`

You can run it locally with Jupyter or upload to Google Colab.

### 4. Run the Streamlit app locally

```bash
cd app
streamlit run app.py
```

The app loads pre-trained models from `../models/` — it does **not** retrain on the fly.

## Streamlit App Tabs

| Tab | Content |
|---|---|
| Executive Summary | Dataset description, prediction task, approach, key findings |
| Descriptive Analytics | Target distribution, age histograms, boxplots, comorbidity analysis, correlation heatmap |
| Model Performance | Comparison table, F1 bar chart, ROC curves, best hyperparameters |
| Explainability & Interactive Prediction | SHAP summary/bar plots, interactive patient risk scorer with waterfall explanation |

## Models Trained

| Model | Tuning |
|---|---|
| Logistic Regression | Baseline (C=1.0) |
| Decision Tree | GridSearchCV over max_depth, min_samples_leaf |
| Random Forest | GridSearchCV over n_estimators, max_depth |
| LightGBM | GridSearchCV over n_estimators, max_depth, learning_rate |
| MLP Neural Network | 2 hidden layers + BatchNorm + Dropout; bonus HP tuning |

All models use `random_state=42`. Train/test split is 70/30, stratified.
